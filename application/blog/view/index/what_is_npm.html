关于npm

npm是什么？

npm是一个让JavaScript程序员分享和复用代码的工具。有了它，JS程序员能高效地管理和发布自己要分享的代码。

安装和更新npm

安装Node.js

如果是用windows或者Mac的话，直接去node.js的官网下载。用的是Linux的话，去找二进制版本。下载完之后要查看npm版本可以在terminal运行：

    node -v

更新npm

    sudo npm install npm -g

更新之后可以再查看一下版本。
